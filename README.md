# TMUX

## Sessions
| Command                       | Explanation          |
| ---                           | ---                  |
| `tmux ls`                     | List sessions        |
| `tmux a -t <session>`         | Attach to session    | 
| `prefix d`                    | Detach from session  |
| `prefix w`                    | List sessions        |
| `prefix )`                    | Next session         |

## Windows 
| Command                       | Explanation          |
| ---                           | ---                  |
| `prefix c`                    | Create new window    |
| `prefix ,`                    | Rename window        |
| `prefix n`                    | Next window          |
| `prefix l`                    | Toggle window        |
| `prefix &`                    | Close window        |

## Panes
| Command                       | Explanation          |
| ---                           | ---                  |
| `prefix x`                    | Close pane           |
